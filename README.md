# Top 250 DSA Question Bank App

A web app to practice the **top 250 DSA (Data Structures & Algorithms) questions** commonly asked in tech interviews.

## Features

- **250 curated DSA questions** covering Arrays, Linked Lists, Trees, Graphs, Dynamic Programming, and more
- **Mark as Complete / Incomplete** — track which questions you've solved (persisted in localStorage)
- **Company Tags** — see which companies asked each question in past interviews (Google, Amazon, Meta, Microsoft, etc.)
- **Topic & Pattern Tags** — each question card shows its topic (e.g., Trees, Graphs) and algorithmic pattern (e.g., DFS, Sliding Window, Two Pointers)
- **Difficulty Badges** — Easy, Medium, Hard color-coded badges
- **Filter & Search** — filter by topic, difficulty, company, completion status, or search by question title
- **Progress Tracking** — visual progress bar showing how many questions you've completed
- **Direct LeetCode Links** — click any question title to open it directly on LeetCode

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Build for Production

```bash
npm run build
```

## Run Tests
